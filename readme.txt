==============
 FontDesigner
==============

FontDesigner is a free (GPL3) tool for creating fonts to be used with arcticOS.
It can save in its own format, or export to the arcticOS format.

INSTALLATION:
Copy the .py file to a directory in your PATH. Optionally remove the ".py" extension.

REQUIREMENTS:
- A monitor, with at lease 800x600 resolution. (Preferably higher, as this app is 800px wide.)
- A window manager or desktop environment
- Python 3.7
- Pygame
